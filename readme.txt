Display a notice to anyone viewing the Dashboard.

== Description ==

This plugin allows you to display a notice to anyone who views your site's Dashboard.

For example, you could alert users to upcoming maintenance or a change in site policy.

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
